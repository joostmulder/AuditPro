AuditPRO Help & Support

Login Help

First Time Users:

If this is your first time using AuditPRO, you will need to create an account through https://www.auditpro.io. Your account must be approved by a registered AuditPRO team prior to use.

Password Reset / Recovery:

To reset your password, use the “Forgot Password” link through https://www.auditpro.io. You will receive reset instructions from accounts@auditpro.io.

Technical Support & Feature Enhancements

For technical support, email support@auditpro.io. Please include your user name, the model of the hardware you are using, and a brief description of the issue. We will respond to technical support requests within 24 hours.

If you’d like to suggest a new feature, or enhancements to existing functionality, we would love to hear from you at support@auditpro.io.

Using AuditPRO

The AuditPRO mobile application is designed as a companion to our web solution. The mobile app is your tool for performing the most efficient & effective store audits. When paired with a Bluetooth printer, AuditPRO also gives you the ability to print reports directly after your audit.

Using AuditPRO for the first time:

Log in & select a store to audit:

  1. Log in to the application using your email address and password
  2. Upon login, you will be asked if you’d like to “Sync to Server” (we strongly recommend syncing prior to each new location’s audit to ensure your device has the latest inventory information)
  3. Select “Audit Store” to find a store to audit. AuditPRO will suggested the nearest location using your device’s GPS location
  4. If you’d like to audit a location that is not shown in GPS-based suggestions, you can filter the list using “ALL CHAINS” at the top of the screen, then sort using the buttons on the bottom:
    1. GPS will sort the filtered list by nearest location
    2. LAST STORE sorts in the order of previous audits (not applicable for first-time use)
    3. NAME sorts alphabetically
  5. Selecting the store to audit will show you additional details about the store and ask you to confirm with “AUDIT THIS STORE”

Begin auditing the store:

  1. Using the attached barcode scanner, scan a UPC
  2. Enter the Retail Price & Sale Price (if applicable)
  3. AuditPRO will default that item to “IN STOCK” (you can change this by clicking)
  4. If you’d like more information about the item, use the (i) button at the top right corner
  5. After confirming the price & status, use the back-arrow in the top left corner, or scan another item, taking you back to the “Select Product” screen

Additional functionality on the “Select Product” screen:

  1. NOTES: enter free-form notes to be stores with your audit
  2. FILTER: hide or display items by status (In-Stock, Void, Out of Stock, none)
  3. Search Text: search by reorder code, brand, sku, or product name

Use “COMPLETE AUDIT” at the bottom of the screen when you’ve finished scanning inventory and setting statuses. From this screen, you can PRINT REORDER REPORT and CLOSE OUT AUDIT, which will store the completed audit on your device until the next time you sync.

Additional Functionality:

After an audit is complete, you will return to the home screen where you can:

  1. Continue an open audit
  2. Start a new audit (if one is not currently open)
  3. Sync to Server (send and receive audit data)
